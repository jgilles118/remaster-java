## The short-hand IF/ELSE statement in a FOR loop

**variable = (condition) ? true result : ELSE result;**

```
for(int i = 0; i<=10; i++){

    String message = "";

    message = (i == 1) ? "You have $" + i + " dollar." : "You have $" + i + " dollars.";

    System.out.println(message);
    //This is perfect!

} 
```
